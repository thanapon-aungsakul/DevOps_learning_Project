# DevOps_learning_Project
A practice repository to learn and apply DevOps concepts for internship preparation.
This project includes environment setup, containerization, service orchestration, networking, and basic CI/CD workflows.

# Overview
This project was created for hands-on DevOps practice during coursework and self-learning.
It demonstrates how to:
  - Set up development environments in containers
  - Build and run services with Docker / Docker Compose
  - Understand service networking and configuration
  - Integrate basic CI/CD workflows using GitHub Actions
  - Explore monitoring workflows (logs, tools)

# Project Structure
DevOps_learning_Project

├── .github/workflows       # CI/CD workflows

├── backend                 # Backend service (example)

├── frontend                # Frontend service (example)

├── docker-compose.yml      # Compose file for multi-service setup

├── README.md
  
# Technologies & Tools
  - Docker & Docker Compose – service containerization
  - Git & GitHub Actions – version control & pipelines
  - Nginx – reverse proxy/load balancing and upstream
  - Monitoring basics – log viewing and simple diagnostics with Prometheus&Grafana
